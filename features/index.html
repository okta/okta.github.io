A fully featured identity and access platform

We’re here to help you spend less time on security and identity, and more time focusing on your application. Implement secure, robust, and scalable identity and access management for your application quickly and easily with Okta.

  • Manage users & groups
  • Secure access
  • Enable single sign-on
  • Directory Integration
  • Integrate with IDPs
  • Create a rich profile

Manage your users and groups

Avoid the hassle of deploying your own LDAP solution. Okta provides a standalone cloud directory and supports the full user lifecycle.

User Management

  • Create and store profile information for any number of users.
  • Customize your user profile to store any number of attributes.
  • Register new users with built-­in on-­boarding workflow.
  • Allow users to update their profiles.
  • Centralize profile data into one authoritative view of the user and keep multiple profile sources synchronized.

Password Management

  • Securely automate setting and updating credentials. Send fully ­customizable account activation and password reset emails, and handle recovery question creation and updates.
  • Use built-­in password management workflows or implement your own.

Groups and Authorization

  • Organize users into groups.
  • Manage access rights to any part of your application with groups or user attributes.
  • Groups and attributes can be defined natively or imported from 3rd party systems such as Active Directory and LDAP.

Secure access to your application

Securely authenticate users and manage their sessions. For extra security, use Okta’s multifactor authentication options.

Authentication

  • Authenticate user credentials against Okta’s cloud directory, or directly against any Active Directory or LDAP solutions
  • For multi-factor authentication, the Okta platform offers Security Questions, SMS-­based one-­time passwords, the Okta Verify mobile app soft token for any mobile device, and supports integration with third party solutions including Verisign VIP, Google Authenticator, and RSA.

Session Management

  • Control session extensions and timeouts whether you are building a web app, thick client app or mobile app.

Enable single sign-on to all your applications

Leverage Okta’s integrations with thousands of applications to implement single sign on for your users. Integrate your own applications to Okta in minutes.

  • Maintain just one set of credentials for each user, across your applications, and third party applications or widgets.
  • Enable your application for single sign-on through standard and non-­standard protocols including SAML (Secure Assertion Markup Language) and OAuth.

Integrate with your partners’ directories

Connect internal or external Active Directory or LDAP user stores to Okta. Include data from any number of directories to create one consolidated directory for your application.

  • Enable external organizations to maintain control over their users, while allowing users to access your application with their existing credentials and profile information.
  • Authenticate against your partners’ or customers’ directories.
  • Drive account creation or deletion, profile synchronization, and authorization directly from an integrated directory.

Integrate with external identity providers

Easily enable your application to authenticate and authorize users against external identity providers through standard federation protocols including SAML and social IDs.

  • Integrate with your partners’ federated identity providers to streamline deployment of your application.
  • Let users log in with their existing social ids to improve the user experience and boost adoption of your application.

Create a rich user profile from multiple data sources

Okta’s user profile is customizable to meet evolving requirements.

  • Customize your user profile to store any number of attributes.
  • Centralize profile data into one authoritative view of the user.
  • Use Okta as a master to keep multiple profile data sources synchronized.
